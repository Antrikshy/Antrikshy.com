About this website

During summer holidays, I was running out of ideas for small programming projects. Besides, I felt a need for landing pages for my software experiments that get finished and shipped. I needed a way to get them to show up on search results without relying on GitHub or reddit posts. I also wanted to add HTML, CSS and JavaScript to my resume. I have always been interested in front-end design but my university courses were not teaching that to me so far.

So I decided to make this website. I used no WYSIWYG tools. I worked on it for months and came up with this.
