About this website

During summer holidays, I was running out of ideas for small programming projects. Besides, I felt the need for landing pages for the projects that I finish and ship. There needed to be a way to get them to show up on search results without relying on GitHub or reddit posts. I also wanted to add HTML, CSS and JavaScript to my resume. I have always been interested in front-end design but I didn't feel as if the university was teaching that to me so far.

So I decided to make this website. I used no WYSIWYG tools and worked on it for months. I wanted to use as many jQuery effects as I tastefully could to make it look professional. I sketched some parts of the website and tried to turn them into reality by manually typing web languages.
