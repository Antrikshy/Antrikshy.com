Story

On reddit, I frequently share content from one subreddit to another if I know a different community that will enjoy it. It's called cross-posting (x-posting) and is encouraged by the reddiquette.

This requires some work. A user opens a new tab, goes to the submission page, switches back, copies the URL, pastes, switches back, copies the title, pastes, adds something like '[x-post from /r/subreddit]', types the other subreddit's name and hits submit. I always found it a bit silly that reddit does not have a button for this on the comments page.

Then it struck me. I could modify the page's CSS to add a button that would automate this. It was also possible to make it sit flush with reddit's design. So I made this extension. It adds a tiny 'xpost' button on a post's comments page, under its title. It goes next to other reddit buttons like 'share'. For people using the Reddit Enhancement Suite browser extension, it fits in snugly with the additional buttons created by that extension.

For the curious

How it works: I use JavaScript to find the button container on the reddit comments page and append a new button to it. It simply links to a pre-formatted submission page, with data from other page elements on the comments page (post title and URL).
