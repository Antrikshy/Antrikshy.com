Story

I decided that GitHub was not an elegant way to distribute any commercial apps or any other code I ship. I needed an elegant website to provide download links from. Making a website would be my foray into the world of web design, which excited me. I also challenged myself to create the website from scratch instead of using commercial tools. I worked for over a month designing the first version before publishing it.

In the process, I learned a lot more than I have from any other side-project. I worked hard tuning the website pixel by pixel until I thought I had it just right.

I plan to keep improving it by using it as a test-ground for any new web technologies that I itch to learn.

For the curious

Learning objectives: I made this to learn the basics of web design. I taught myself HTML, CSS, jQuery, and using APIs in the process.
