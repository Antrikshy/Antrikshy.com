Story

On Windows, it's pretty standard to expect a web browser to have a reopen closed tabs feature. Usually,this is done by the Ctrl+Shift+T keyboard shortcut. This is common on Mac as well. Most browsers do this using Cmd+Shift+T. Except Apple's Safari. Pressing Cmd+Z to undo reopens up to one closed tab and doesn't always work.

I have seen a lot of clamoring about this feature on the Internet. It doesn't look like anyone has ever made a simple, no-frills, standalone extension to implement this feature. So I took it upon myself to do it.

For the curious

How it works: Using the Safari API, I listen for the signal the browser sends when a tab is about to close. Before it closes, I save the URL of the target tab in an array. As tabs are closed, the URLs pile up. Cmd+Shift+T pops them out and opens new tabs with this URL.
