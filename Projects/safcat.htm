Story

While visiting family in Cambodia, I had to endure a torturously slow Internet connection. While there was no viable solution for slow-downloading websites and sluggish video streaming, there was one for animated gifs.

While gifs are increasingly popular on the Internet right now, they load slowly in general. There is a solution, gfycat - a tool that compresses and converts gifs to HTML5 video. But using that requires manually making it fetch every gif I want to watch, which takes about the same time but more effort. There were no extensions for the browser of my choice (Safari) that made this easier.

I decided to give browser extensions a try. After reading piles of JavaScript and Apple documentation, I came up with safcat - a toolbar button that uploads any open gif to gfycat and plays them back immediately.

For the curious

Learning objectives: I made this mostly for my own convenience, but also to learn JavaScript, to get an idea about browser extension APIs and to see how JSON is used, since I haven't learned these in university so far.

How it works: When you press the button, JavaScript code inside the extension sends a certain request to gfycat, which includes your active tab's address. gfycat attempts to fetch the content from your URL. If it is a gif, it is fetched and you are redirected to the HTML5 video version of it. If it fails for whatever reason, you get an error message.
