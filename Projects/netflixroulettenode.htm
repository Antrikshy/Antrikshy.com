About

This NetflixRoulette library is a simple module I wrote to learn Node.js.

The module allows you to interface with the very simple Node.js API without messing around with HTTP request libraries in your Node.js project. The module supports all the endpoints that NetflixRoulette supports and I hope to keep it updated.

Currently, it exports functions to allow users to interface with the title, actor and director endpoints in the API.

Instructions

Install using npm install netflixroulette.

Require using

var NetflixRoulette = require('netflix-roulette');

Now you can interface with the three endpoints NetflixRoulette currently provides:

NetflixRoulette.title('Tucker & Dale vs. Evil', callbackFunc(error, data), 2010);
[year parameter optional]

NetflixRoulette.actor('Keira Knightley', callbackFunc(error, data));

NetflixRoulette.director('Christopher Nolan', callbackFunc(error, data));

NetflixRoulette does not require any sort of authentication because its API is open. error is only set when there is a problem requesting data from the server. Any errors in finding the data will be reported in the data variable with error == null.

Refer to NetflixRoulette's API guide for details on returned data.
