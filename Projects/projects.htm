Projects.

reddit bots
I have experimented a bit with Python scripts that autonomously post on reddit. reddit bots are common. Some are merely funny while some have utility. I have made one bot of the former kind and have some ideas for making some of the other.

safcat
A Safari browser extension written in JavaScript made for people who watch animated gifs frequently. It adds a toolbar button that instantly uploads an open gif to gfycat, an online service that compresses it and converts it to HTML5 video. Thus, painfully slow loading gifs are played back much quicker with one click.

Antrikshy.com
This website is my first, very experimental project written in HTML5, CSS3 and jQuery. It was written completely by hand. The only tools I used were Sublime Text and browser developer tools. I have tried my best to make it work well across browsers and platforms. It currently uses the about.me API and is hosted on Amazon S3.

Twitter RT Analyzer
A small command line tool that searches Twitter for keywords, gets results and computes whether the users in the results are interactive on the network. It does this by going through their past tweets and calculating how many of them were retweets or replies.

Unnamed homepage website
I am working on a website that is meant to be a simple start page for your web browser. It displays the time, date, current weather and some advice for you if you're leaving home, according to the current weather.

Scan and Reply Framework
I am working on a full-featured scan-and-respond style reddit bot engine. It is a user-friendly command line program that allows a user to connect a reddit account and run a highly-configurable bot that scans for a set of phrases and replies with a set comment.
