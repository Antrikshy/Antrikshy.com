NetflixRoulette Node.js Wrapper
A Node.js module that makes it easy to communicate with the very simple NetflixRoulette API without reinventing the wheel. NetflixRoulette allows you to look up titles, actors and directors on Netflix Instant.

reddit2Kindle
A Python command line tool that works with Amazon's KindleGen script to quickly compile a Kindle book from the top text posts in a specified subreddit for a specified time period.

safcat
A Safari browser extension made for people who watch animated gifs frequently. It adds a toolbar button that uploads an open gif to gfycat, an online service that compresses it and converts it to HTML5 video. Painfully slow loading gifs are played back much quicker with one click.

x-poster
An extension for Safari on Mac that adds a small xpost button under reddit posts on their comments pages. Clicking this lets you easily share links across subreddits (known as cross-posting, or x-posting). It redirects you to a new submission page with the title and URL fields already filled.

RecoverTabs
An extension that adds the much-wished-for Cmd+Shift+T feature to Safari on Mac. In other browsers, Cmd+Shift+T reopens several previously closed tabs. There is a version of this integrated into the undo (Cmd+Z) functionality in Safari, but it only opens one tab and doesn't always work. RecoverTabs lets you intuitively reopen multiple tabs.

Antrikshy.com
This website is my first, very experimental project written in HTML5, CSS3 and jQuery. It was written completely by hand. The only tools I used were Sublime Text and browser developer tools. I have tried my best to make it work well across browsers and platforms.

Twitter RT Analyzer
A small command line tool that searches Twitter for keywords, gets results and computes whether the users in the results are interactive on the network. It does this by going through their past tweets and calculating how many of them were retweets or replies.

Unnamed homepage website
I am working on a website that is meant to be a simple start page for your web browser. It displays the time, date, current weather and some advice for you if you're leaving home, according to the current weather.

reddit bots
I have experimented a bit with Python scripts that autonomously post on reddit. reddit bots are common. Some are merely funny while some have utility. I have made one bot of the former kind and have some ideas for making some of the other.
